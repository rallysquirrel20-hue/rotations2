## UI/UX — Header Redesign & Responsive Layout

**Type**: Change / Refactor
**Files modified**:
- `app/frontend/index.html` (line 7)

**What was done**:

### 1. Browser Tab Title
- Changed `<title>` from "Vite + React + TS" to "Rotations"

### 2. Cross-Basket Returns Bar Chart Hover Fix
- **Bug**: After adding left padding to the cross-basket returns bar chart, hover highlighting was offset — cursor highlighted bars to the right of actual cursor position
- **Root cause**: The `handleMouseMove` handler used hardcoded `{ left: 60, right: 50 }` padding while the rendering code (`drawBarChart`) computed dynamic padding based on label widths
- **Fix**: Added `barGeoRef` to store the actual bar geometry (padLeft, padRight, barW, gap, n) during rendering, and the hover handler now reads from this ref instead of recalculating with wrong values

### 3. Responsive Header — Sidebar Resize
- **Problem**: Dragging the left sidebar pane wider caused header buttons/toggles to overflow off-screen; pulling it back left a gap on the right
- **Solution**: Converted `.main-header` from CSS Grid to Flexbox layout. The 6 action buttons (Basket Returns, Analogs, Backtest, Intrabasket, Signals, Export) and date/universe controls are fixed-size (`flex: 0 0 auto`) and right-aligned via `margin-left: auto`. Only the toggles area was flexible.

### 4. Indicator Toggles → Dropdown Menu
- **Problem**: With many basket toggles (Breadth%, Breakout%, Correlation%, RV%, Constituents), the inline checkboxes got clipped when the header compressed
- **Solution**: Replaced inline toggle checkboxes with an "Indicators" dropdown button. Click opens a dropdown overlay with all toggle checkboxes, works at any header width

### 5. Ticker/Basket Search Box Restyled
- Restyled the search/title box to match the other header buttons (same 12px uppercase bold font, 32px height, border style)
- Search dropdown now appears immediately on click (not just after typing), showing all items with category filter tabs (All, ETFs, Themes, Sectors, Industries, Tickers) — same behavior as the backtest panel's picker

### 6. Header Left Stack Layout
- Grouped the ticker/basket search box and Indicators dropdown into a vertical stack (`header-left-stack`) on the left side
- Both buttons are 120px wide, visually mirroring the 3×2 button grid on the right

### 7. Dropdown Z-Index Fix
- **Bug**: Search dropdown and Indicators dropdown were clipped by the header boundary, not overlapping the chart
- **Fix**: Removed `overflow: hidden` from `.main-header` and `.search-container` so `z-index: 1000` dropdowns render on top of the chart

### 8. Chart Resize on Sidebar Drag
- **Bug**: TradingView lightweight-charts only listened to `window.resize` events — dragging the sidebar didn't trigger a window resize, so the chart didn't reflow
- **Fix**: Added a `ResizeObserver` on `wrapperRef` in TVChart.tsx so charts resize whenever their container changes dimensions

**Result**: Success
**Follow-up needed**: None — header is responsive, chart resizes correctly, all dropdowns overlay properly
