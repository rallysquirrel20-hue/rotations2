# Session 2026-04-20 — Distributions panel major expansion

Commit: `85ef5d7` (plus this meta update)
Scope: `app/backend/main.py` + `app/frontend/src/components/DistributionsPanel.tsx` only. Consumer-side — no producer/schema changes.

## Phases 1–5 (all shipped in 85ef5d7)

- **Phase 1 — `_label_rotation_context` helper.** Per-bar metrics for UpRot/DownRot/Breakout/Breakdown: Time (bar count), Range (favorable extension as decimal, min-low for bearish types), Change (signed, entry-open normalized). Indices 0..3. Expanding-window percentile ranks per metric (strictly-prior pool, midrank ties). Extreme_0..3 (peak high or trough low) for pattern detection. Active rotations use running cummax/cummin.
- **Phase 2 — rot JSON filter + collapsible UI.** Endpoint accepts a `rot` JSON spec with threshold/percentile/trend/pattern per rotation type. Legacy flat threshold+percentile formats still accepted (applied to index 0).
- **Phase 3 — horizon selector + forward_paths.** Horizon ∈ {1, 5, 21, 63, 252, 'rotation'}. Rotation mode holds until Trend flips (point return only, no path). Fixed horizons return up to 500 downsampled match paths (≤127 pts when horizon > 126).
- **Phase 4 — tercile-split forward-paths chart.** Sorted by final return; top/middle/bottom third drawn as green/gray/red mean lines with ±1σ filled bands. Sortable match-list sidebar; hover overlay for single-path inspection. 560 px tall.
- **Phase 5 — CURRENT CONTEXT card.** Summary header (ticker/date/close/active rotation+regime/position vs target), indicators strip (RV ann + %ile, H/L/P react + %ile, upper/lower, and breadth/breakout/corr for baskets), rotation table with Pattern column color-coded HH/LH/HL/LL.
